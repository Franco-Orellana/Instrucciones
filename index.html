Modo de instalar correctamente el software

Para que no exista ningún problema y todo funcione correctamente, debe realizar la instalación de los diferentes componentes de la siguiente manera:

  • En primer lugar, instalar el software “SQL Server Express”.
  • En segundo lugar, instalar el software “.NET Desktop Runtime”.
  • En tercer lugar, ajustar la fecha, horario y región.
  • Por último, instalar el software “TurnoWave”.

En caso de tener problemas:

  • Solucionar problemas con Base de datos.
  • Solucionar problemas con Antivirus de Windows.

Instalación de SQL Server Express

SQL Server es un sistema de gestión de bases de datos relacionales (RDBMS) desarrollado por Microsoft. Permite almacenar y recuperar datos según lo solicitado por otras aplicaciones de software, ya sea en la misma computadora o a través de una red. Este software se utilizará para crear y/o administrar la base de datos necesaria para almacenar los turnos y los datos que tiene aparejados.

En caso de usar un Sistema Operativo Windows 10 o posterior, deberá descargar la última versión disponible (2022): Descargar

El instalador en el sitio web.

Una vez descargado el instalador correspondiente, debe buscarlo en la carpeta donde se guardan las descargas para luego ejecutarlo. Luego de ejecutar el archivo, ósea hacer doble click sobre este, observará una ventana donde deberá pulsar la opción “Sí”.

El instalador descargado.

Le aparecerá esta ventana, que dará comienzo al proceso de instalación, aquí deberá seleccionar la opción “Personalizado”:

En esta ventana tendrá que cambiar la “UBICACIÓN DE LOS MEDIOS”, por defecto está establecido como “C:\SQL2022” pero debe cambiarlo a “C:\Program Files\Microsoft SQL Server”, luego de hacer este cambio debe que presionar el botón “Instalar”, el cual dará comienzo a la descarga de los archivos necesarios para la instalación.

Finalizada la descarga de los paquetes de instalación, aparecerá la siguiente ventana donde debe seleccionar la primer opción, osea, “New SQL Server standalone installation on add features to an existing installation”.

En esta primera ventana donde comenzará a instalarse el software, simplemente tendrá que pulsar el botón “Next”.

En el apartado “License Terms”, tendrá que aceptar los términos de la licencia, pulsando sobre “I accept the license terms and” y luego sobre el botón “Next”.

En el apartado “Azure Extension for SQL Server”, tendrá que quitar el “check” que viene por defecto sobre “Azure Extension for SQL Server” que contiene una leyenda a su derecha, ya que no es necesario y luego sobre el botón “Next”.

En el apartado “Feature Selection”, tendrá hacer click sobre la opción “LocalDB”, en caso de no estar seleccionada, luego el botón “Next” para continuar.

En el apartado “Instance Configuration”, tendrá que elegir la opción “Default instance” y luego hacer click sobre el botón “Next”.

En el apartado “Server Configuration”, solamente tendrá hacer click sobre el botón “Next”.

En el apartado “Database Engine Configuration”, tendrá que elegir la opción “Windows authentication mode” y luego hacer click sobre el botón “Next”.

Por último, en el apartado “Installation Progress”, simplemente debe esperar a que se concrete la instalación.

Una vez se haya concretado la instalación simplemente debe cerrar la ventana pulsando sobre el botón “Close”. De igual manera, la ventana denominada “SQL Server Installation Center”, en caso de estar abierta.

Instalación de software “.NET Desktop Runtime”

Debe descargar el software desde la página web oficial de Microsoft: Descargar

Luego debe buscar el instalador en la carpeta donde se guardan las descargas para ejecutarlo, ósea hacer doble click sobre este, observará una ventana donde deberá pulsar la opción “Sí”.

El instalador es este.

Se abrirá la siguiente ventana, donde comenzará el proceso de instalación. Solamente tendrá que hacer click sobre “Install” para comenzar la instalación.

Luego de concertar el paso anterior, le mostrará la ventana “Setup Progress”, donde se instalará el mismo, y por último se mostrará la ventana “Installation was successful” que indicará el fin del proceso de instalación.

Base de datos

Una de las causas de que no se inicie correctamente el programa es por un error en creación de la base de datos necesaria para almacenar los datos correspondientes.

Para corregir este problema debe hacer los siguientes pasos:

1°: Descargar el archivo sql que configura la base de datos. Descargar

2°: Pulsar las teclas de Windows + R de su teclado, al mismo tiempo. Esto le abrirá una ventana de "Ejecutar", donde debe escribir la palabra "cmd" y presionar el botón "Aceptar".

3°: Debería aparecer la siguiente ventana.

4°: En esta ventana de "Simbolo del sistema", debe escribir el comando sqlcmd -S localhost -E -Q "SELECT name FROM sys.databases" y luego para que se ejecute debe pulsar "Enter".

Esto le permitirá verificar si existe la base de datos necesaria, la cual se llama "GestionTurnos".

5°: Por último, si no existe la base de datos, debe escribir el comando sqlcmd -S localhost -E -i C:\Users\NombreDelUsuario\Downloads\script.sql y luego para que se ejecute debe pulsar "Enter".

Luego de seguir los pasos anteriores, si el resultado de la operación es igual al mostrado en la imagen anterior entonces la base de datos se creó de manera correcta.

Intente abrir el programa y verifique su correcto funcionamiento.

Antivirus de Windows

En caso de experimentar este tipo de problemas, debe actuar de la siguiente manera:

Al ejecutar el instalador denominado “TurnoWaveSetup-x64”, puede que le aparezca esta ventana, usted debe pulsar sobre el enlace “Más información”.

Luego de hacer lo antes indicado, observará más información sobre el instalador como el nombre de la aplicación y del editor, acompañado de un segundo botón para seleccionar. Para comenzar el proceso de instalación debe pulsar sobre el botón “Ejecutar de todas formas”.

Ajustar fecha y hora de Windows

Para no tener problemas de este tipo, debe configurar la fecha y horario:

1°: Diríjase al margen inferior derecho de la pantalla, donde encontrará el horario y la fecha, aquí debe posicionarse sobre este y hacer click derecho sobre el mismo. Aparecerán opciones, debe elegir “Ajustar fecha y hora”.

2°: En esta ventana llamada “Fecha y hora”, debe ir hacia abajo, donde encontrará el apartado “Configuración relacionada”. Debe seleccionar “Fecha, hora y formato regional”.

3°: En esta ventana llamada “Región”, debe cambiar tanto “País o región” como “Formato actual” por “Argentina” y “Español (Argentina)” respectivamente.
