Problemas con inicio del programa

Una de las causas de que no se inicie correctamente el programa es por un error en creación de la base de datos necesaria para almacenar los datos correspondientes.

Para corregir este problema debe hacer los siguientes pasos:

1°: Descargar el archivo sql que configura la base de datos. Descargar

2°: Pulsar las teclas de Windows + R de su teclado, al mismo tiempo. Esto le abrirá una ventana de "Ejecutar", donde debe escribir la palabra "cmd" y presionar el botón "Aceptar".

3°: Debería aparecer la siguiente ventana.

4°: En esta ventana de "Simbolo del sistema", debe escribir el comando "sqlcmd -S localhost -E -Q "SELECT name FROM sys.databases", sin las comillas. Esto le permitirá verificar si existe la base de datos necesaria, la cual se llama "GestionTurnos".

5°: Por último, si no existe la base de datos, debe escribir el comando "sqlcmd -S localhost -E -i C:\Users\NombreDelUsuario\Downloads\script.sql", sin las comillas.

Luego de seguir los pasos anteriores, si el resultado de la operación es igual al mostrado en la imagen anterior entonces la base de datos se creó de manera correcta.

Intente abrir el programa y verifique su correcto funcionamiento.
